# Supermarket Receipt test-list

- [ ] Have a name and a price
- [ ] Allow decimal prices
- [ ] Throw an exception when price is negative

## Product quantity
- [ ] Store product and quantity
- [ ] Throw an exception when quantity is zero or negative

## Shopping cart
- [ ] Be empty when created
- [ ] Add a single product

## Receipt

- [ ] Calculate subtotal correctly
- [ ] Show zero total when cart is empty
- [ ] Include total price
- [ ] List items in the order they were added
